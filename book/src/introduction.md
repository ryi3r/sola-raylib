# sola-raylib

The long-form documentation for
[sola-raylib](https://github.com/brettchalupa/sola-raylib), the actively
maintained Rust binding to [raylib](https://www.raylib.com/). API reference docs
live on [docs.rs](https://docs.rs/sola-raylib); the project README has the
install instructions. This book covers the topic-by-topic recipes and
walkthroughs that fit in neither.

View the crates:

- [crates.io/sola-raylib](https://crates.io/crates/sola-raylib) - safe wrappers
  and ergonomic APIs
- [crates.io/sola-raylib-sys](https://crates.io/crates/sola-raylib-sys) - raw
  FFI bindings for Raylib C

**The book is currently a very early work in progress.**
